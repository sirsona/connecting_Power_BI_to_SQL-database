# How to Connect Power BI to a PostgreSQL Database (Localhost and Aiven Cloud)

This repository contains a guide explaining how to connect Power BI to a
PostgreSQL database both locally and using Aiven PostgreSQL (cloud).

## What the Article Covers

- Introduction to Power BI and business intelligence
- Connecting Power BI to a local PostgreSQL database
- Connecting Power BI to Aiven PostgreSQL
- Installing the CA SSL certificate for secure connections
- Loading tables (`customers`, `products`, `sales`, `inventory`) into Power BI
